## Algorithms

### Searching Algorithms
#### Binary Search
**Space Complexity**

`Worst complexity:` O(1)
`Best complexity:` O(1)

**Time Complexity**

`Worst complexity:` O(log n)
`Average complexity:` O(log n)
`Best complexity:` O(1)

**Space Complexity**

`Worst complexity:` O(1)
`Best complexity:` O(1)

**Time Complexity**

`Worst Case`: O(n * n)
`Best Case`: O(n)
`Average Time`: O(n * n)
